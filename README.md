##  Bill of Materials (BOM)


- 1x XIAO RP2040 controller
- 15x Mx-switches
- 16x 1N4148 diodes
- 1x Rotary encoder with switch
- 2x SK6812MINI RGB LED
- 1x Pcb
- 1X 3d printed case
- 4x Screws + standoffs for case assembly
